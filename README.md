# Bruce Lee Buddha

## Project Description
This project is dedicated to the martial arts philosophy of Bruce Lee combined with the spiritual teachings of Buddha. It aims to provide a comprehensive understanding of their quotes and principles.

## Features
- **Quote Database**: A collection of famous quotes from Bruce Lee and Buddha.
- **Progressive Web App (PWA)**: Fully functional PWA for easy accessibility.
- **Node.js Server**: Backend server for handling requests and serving data.
- **Android App**: A mobile application that integrates project features.

## Installation Instructions
### Progressive Web App (PWA)
1. Clone the repository.
   ```sh
   git clone https://github.com/vsevolobo/bruce-lee-buddha.git
   ```
2. Navigate to the project directory.
3. Open the index.html file in a web browser, or deploy it using any suitable server.

### Node.js Server
1. Ensure you have Node.js installed.
2. Clone the repository.
3. Navigate to the server directory.
   ```sh
   cd server
   ```
4. Install the required dependencies.
   ```sh
   npm install
   ```
5. Start the server.
   ```sh
   npm start
   ```

### Android App
1. Clone the repository.
2. Open the project in Android Studio.
3. Build and run the app on your device or emulator.

## Build Variants
- **Development**: Use for testing and development purposes.
- **Production**: Optimized version for deployment.

## Setup Guides
- Follow the respective instructions for setting up PWA, Node.js, and Android App.
